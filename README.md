# cropin-website

Static mirror of [cropin.com](https://www.cropin.com/) — full HTML, CSS, JS, images, and animations as captured via wget.

## Local preview

```bash
python3 -m http.server 8888
```

Open [http://localhost:8888](http://localhost:8888).

## Deploy

Static site on Vercel — no build step. Root directory is the site.
